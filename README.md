# Airbnb Sales & Revenue Analysis Dashboard | Tableau Project

1. Project Description / Purpose

This Tableau dashboard provides an interactive analysis of Airbnb listing performance using a real-world Airbnb dataset. The dashboard helps users understand pricing trends, revenue patterns, bedroom-wise pricing, and geographical distribution of listings across different ZIP codes.
The goal of this project is to transform raw Airbnb data into meaningful business insights that can help property owners, investors, and analysts make data-driven decisions regarding pricing strategies and revenue optimization.

2. Tech Stack
- Tableau Public/Desktop – Data Visualization & Dashboard Development
- Microsoft Excel / CSV – Data Source
- Data Cleaning & Preparation – Excel/Tableau
- Mapbox Integration – Geographic Visualization
- GitHub – Project Documentation & Portfolio Showcase

3. Data Source
Dataset: Airbnb Listings Dataset.

The dataset contains information related to Airbnb properties, including:
- Listing ID
- ZIP Code
- Number of Bedrooms
- Price per Listing
- Calendar Date
- Revenue Information
- Property Location Data

4. Features / Highlights

# Business Problem
Airbnb hosts and property managers need insights into how pricing varies across locations and property sizes. Understanding these trends can help maximize revenue and improve listing performance.

# Dashboard Components

1. Average Price Per Bedroom
- Displays the average Airbnb price based on the number of bedrooms.
- Helps identify how property size influences pricing.
- Shows a clear upward trend in price as bedroom count increases.
2. Price Per ZIP Code (Map View)
- Interactive geographic visualization of Airbnb prices.
- Allows users to identify high-value and low-value locations.
- Provides spatial insights into pricing distribution.
3. Price by ZIP Code (Bar Chart)
- Compares average prices across different ZIP codes.
- Enables ranking and comparison of neighborhoods.
- Highlights premium and budget-friendly areas.
4. Revenue for Year Trend Analysis
- Displays revenue performance throughout the year.
- Helps identify seasonal demand patterns.
- Useful for forecasting and business planning.
5. Distinct Count of Bedroom Listings
- Shows the distribution of listings by bedroom count.
- Helps understand inventory availability across property types.
- Supports market segmentation analysis.

# Key Insights
- Properties with more bedrooms generally command higher prices.
- Certain ZIP codes generate significantly higher average prices than others.
- Revenue remains relatively stable throughout the year with noticeable peak periods.
- One-bedroom properties represent the largest share of Airbnb listings.
- Geographic analysis reveals pricing differences across neighborhoods.

# Outcome
This dashboard enables users to:
- Analyze Airbnb pricing trends.
- Compare property performance across locations.
- Identify high-revenue opportunities.
- Understand bedroom-wise market demand.
- Make informed pricing and investment decisions.
